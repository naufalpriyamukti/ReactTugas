Frontend (Sisi Klien)

React.
○ Manajemen State: Gunakan hooks React (seperti useState, useEffect,
useContext) untuk mengelola state aplikasi.
○ Routing: Implementasikan navigasi antar halaman menggunakan React Router.
○ Komunikasi API: Lakukan pemanggilan ke backend API (fetch/GET, POST, PUT,
DELETE).
○ Styling: Berikan gaya dan tata letak yang bersih, responsif, dan presentabel. Anda
bisa menggunakan CSS biasa atau framework UI seperti Tailwind CSS.
